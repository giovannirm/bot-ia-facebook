Para compilar el proyecto

1.- Descargar Anaconda(servirá como entorno virtual)
2.- Descargar MongoDb(será la BD del proyecto)

Entrar al cmd

3.- Crear entorno(C:\Users\usuario>)
conda create -n nombre_del_entorno python=3.6 <- version de python a instalar en el entorno

4.- Activar entorno(C:\Users\usuario>)
activate nombre_del_entorno

5.- Entrar a la ruta del proyecto((chat) D:\BOTS-PYTHON\botFacebook>)

6.- Instalar los siguientes paquetes

Flask==1.1.2
requests==2.24.0
nltk==3.5
numpy==1.18.5
tflearn==0.3.2
tensorflow==1.14.0
Django==2.2.7
djongo==1.3.3

Repetir ese ejemplo para cada paquete
Ejm: (chat) D:\BOTS-PYTHON\botFacebook>pip install Flask==1.1.2

DATO:
-----------------------------------------------------------------
Para saber que paquetes tenemos instalados en nuestro entorno
(chat) D:\BOTS-PYTHON\botFacebook>pip freeze
-----------------------------------------------------------------

7.- Ejecutar el siguiente comando,
    es para que en la carpeta petWeb/migrations cree el modelo de nuestra bd
(chat) D:\BOTS-PYTHON\botFacebook>python manage.py makemigrations

8.- Ejecutar el siguiente comando,
    es para que cree la bd junto con las colecciones en mongodb
(chat) D:\BOTS-PYTHON\botFacebook>python manage.py migrate

9.- Ejecutar el siguiente comando,
    es para ejecutar el proyecto
(chat) D:\BOTS-PYTHON\botFacebook>python manage.py runserver
